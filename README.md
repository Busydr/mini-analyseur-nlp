# Mini Analyseur NLP

Un programme Python qui analyse un fichier texte et produit un rapport complet.

## Ce que le programme fait

- Lit un fichier texte
- Nettoie le texte (ponctuation, minuscules)
- Compte le nombre total de mots
- Compte le nombre de mots uniques
- Trouve les 3 mots les plus fréquents
- Calcule la longueur moyenne des mots
- Compte le nombre de phrases
- Sauvegarde le rapport dans un fichier rapport.txt

## Technologies utilisées

- Python 3
- Programmation orientée objet (Classes)
- Manipulation de fichiers

## Comment utiliser le programme

1. Clone le dépôt
2. Place ton fichier texte dans le même dossier
3. Exécute le programme :
python MiniAnalyseurNLP.py
4. Entre le nom du fichier quand le programme le demande
5. Le rapport sera sauvegardé dans rapport.txt

## Exemple de résultat

=== RAPPORT D'ANALYSE ===

Nombre total de mots : 728

Nombre de mots uniques : 348

Top 3 mots les plus fréquents :
  la : 31
  de : 30
  une : 25
  
Longueur moyenne des mots : 5.73

Nombre de phrases : 45
